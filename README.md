# aoc-zig
Advent of Code solutions in Zig

## Project setup

Dependencies:
- zig 0.13.0

Setup git hooks:
```bash
git config core.hooksPath .githooks
```

## Running the day solution
```bash
zig run main.zig -- <year> <day>
```
Example
```bash
zig run main.zig -- 2024 1
```
Runs both part 1 and part 2 of the solution.
